# Data Analysis & Physics Simulations Portfolio

This repository contains a collection of data analysis projects implemented in Python. 

The primary focus of these projects is to translate mathematical and physical models into clean, efficient, and well-documented code. I emphasize the use of vectorization and modern data science libraries to optimize performance.

## 🛠️ Tech Stack & Skills
* **Languages:** Python
* **Libraries:** NumPy, Matplotlib
* **Concepts:** Monte Carlo Methods,Data Visualization, Algorithm Optimization (Vectorization)

## 📂 Project Modules

* **[01_pi_approximation](./01_pi_approximation)**
  * *Description:* A vectorized Monte Carlo simulation to estimate the mathematical constant Pi.
  * *Focus:* Fundamental random sampling, computational efficiency using NumPy, and data visualization.

* *(More modules like Random Walks, N-Dimensional Integration, and the Ising Model are currently in development...)*
